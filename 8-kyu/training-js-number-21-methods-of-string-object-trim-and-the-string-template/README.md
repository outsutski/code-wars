Usage is very simple:</p><pre><code class="language-javascript"><span class="cm-variable">stringObject</span>.<span class="cm-property">trim</span>()</code></pre><p>Its role is very simple too: remove the whitespace at the edge of the string. example:</p><pre><code class="language-javascript"><span class="cm-keyword">let</span> <span class="cm-def">str</span><span class="cm-operator">=</span><span middle of the string will not be removed.</p><pre><code class="language-javascript"><span class="cm-keyword">let</span> <span class="cm-def">str</span><span class="cm-operator">=</span><span Using double quotes <code>"</code> or single quotes <code>'</code>, we get a fixed string value. Use the backtick <code>`</code>, we are defining a string template. </p><p>We can use <code>${variable}</code> insert a variable into string template. like this:</p><pre><code ${</span><span class="cm-variable-2">a</span><span class="cm-operator">+</span><span class="cm-variable-2">b</span><span class="cm-string-2">}</span><span class="cm-string-2">`</span>);}<span <span class="cm-number">7</span></code></pre><p>Isn't it interesting?
